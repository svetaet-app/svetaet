1. Общие положения

Настоящая Политика конфиденциальности описывает, какие данные обрабатывает приложение «Светает. Умное пробуждение.» («Приложение»), для каких целей они используются, кому могут передаваться и как пользователь может управлять обработкой данных.

Разработчик и оператор данных Приложения: Ефременко Никита Александрович.

Контакт по вопросам обработки данных: svetaet.support@gmail.com.

Правовые основания обработки данных зависят от конкретной функции: действия пользователя при настройке Приложения и использовании его функций, пользовательское соглашение, согласие пользователя на обработку данных для рекламы и технической аналитики, а также требования применимого законодательства.

2. Какие данные хранятся локально

Приложение хранит на устройстве пользователя:

  • настройки будильников;
  • выбранные задания пробуждения;
  • QR-данные для QR-задания;
  • настройки темы, громкости, повторов и напоминаний;
  • согласия пользователя;
  • статистику пробуждений.

Эти данные используются для работы будильника, заданий и статистики. Они хранятся локально на устройстве и не передаются на серверы разработчика.

3. Реклама

Приложение может показывать рекламные блоки на некритичных экранах: в списке будильников, статистике и настройках.

Реклама не показывается на экране звонка будильника, во время выполнения задания пробуждения, на экране успешного завершения и в важных шагах настройки разрешений.

Для показа рекламы используется Рекламная сеть Яндекса. Пользователь может отключить персонализацию рекламы в настройках Приложения; это не обязательно отключает все рекламные блоки, но ограничивает использование согласия на персонализацию. Если рекламный блок не загрузился, место под него не показывается.

Рекламный SDK может обрабатывать технические сведения об устройстве и приложении, сетевые сведения, рекламный идентификатор Android, данные о загрузке, показе и взаимодействии с рекламой, а также диагностические данные. Состав данных зависит от фактической конфигурации SDK и требований магазина приложений.

4. Техническая аналитика

При согласии пользователя Приложение использует AppMetrica для технической аналитики, диагностики стабильности и улучшения качества.

AppMetrica может обрабатывать:

  • технические сведения об устройстве, ОС и версии приложения;
  • сведения о запуске и использовании приложения;
  • продуктовые события без пользовательских текстов: прохождение начальной настройки, открытие экранов, настройка будильника по категориям, тип задания, результат выполнения, откладывание сигнала и проверка после подъёма;
  • события стабильности, сбои и диагностические данные;
  • идентификаторы устройства или приложения, если они доступны SDK.

Приложение не передаёт в AppMetrica точное время будильника, текст напоминания, QR-данные, выбранную мелодию, контакты, SMS, фотографии, микрофон и точное местоположение.

Пользователь может отключить отправку аналитики в настройках Приложения. Статистика будильников внутри приложения продолжит работать локально.

5. Разрешения Android

Приложение запрашивает разрешения только для заявленных функций:

  • уведомления — для активного будильника и службы активного сигнала;
  • точные будильники — для срабатывания в выбранное время;
  • экран поверх блокировки — чтобы показать экран звонка будильника;
  • служба активного сигнала — для воспроизведения звука и вибрации активного будильника;
  • камера — только для сканирования QR-кода в QR-задании;
  • физическая активность — только для подсчёта шагов в задании «Шаги»;
  • датчики движения — только для задания «Встряхивание»;
  • рекламный идентификатор Android — для показа рекламы, измерения рекламных событий, диагностики рекламных блоков и предотвращения злоупотреблений при фактическом показе рекламы;
  • интернет и состояние сети — для рекламы, технической аналитики и диагностики SDK;
  • настройки оптимизации батареи — по выбору пользователя для повышения надёжности будильника.

Приложение не запрашивает контакты, SMS, журнал звонков, микрофон, точное местоположение и доступ к фотографиям пользователя.

6. Передача третьим лицам

Разработчик не продаёт пользовательские данные.

Данные могут обрабатываться третьими лицами только через используемые SDK:

  • AppMetrica — техническая аналитика при согласии пользователя;
  • Рекламная сеть Яндекса — показ рекламы, персонализация рекламы при согласии пользователя, подсчёт показов и диагностика рекламных блоков.

Обработка данных этими SDK регулируется политиками соответствующих поставщиков:

  • AppMetrica: https://appmetrica.yandex.ru/docs/ru/sdk/android/analytics/android-operations
  • Yandex Mobile Ads: https://yandex.ru/legal/mobileads_sdk_agreement/

7. Хранение и удаление данных

Локальные данные хранятся на устройстве до удаления приложения, очистки данных приложения в настройках Android или сброса статистики внутри Приложения.

Данные, которые обрабатываются SDK Яндекса, хранятся по правилам соответствующих сервисов Яндекса.

8. Права пользователя

Пользователь может:

  • отключить техническую аналитику;
  • отключить персонализацию рекламы;
  • отозвать разрешения Android в системных настройках;
  • сбросить статистику пробуждений;
  • удалить Приложение и локальные данные;
  • обратиться по вопросам обработки данных, получения информации, уточнения или удаления данных на svetaet.support@gmail.com.
