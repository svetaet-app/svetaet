Политика конфиденциальности приложения «Светает»

Дата вступления в силу: 17 мая 2026 года

Настоящая Политика конфиденциальности описывает, какие данные обрабатывает приложение «Светает. Умное пробуждение.».

Разработчик приложения: Ефременко Никита Александрович.
Контакт по вопросам обработки данных: svetaet.support@gmail.com.

1. Какие данные хранятся локально

Приложение может хранить на устройстве пользователя:

  • настройки будильников;
  • выбранные задания пробуждения;
  • QR-данные для QR-задания;
  • настройки темы, громкости, повторов и напоминаний;
  • согласия пользователя;
  • статистику пробуждений.

Эти данные используются для работы будильника, заданий и статистики. Они хранятся локально на устройстве и не передаются на серверы разработчика.

2. Реклама

Приложение может показывать рекламные блоки на некритичных экранах: в списке будильников, статистике и настройках.

Реклама не показывается на экране звонка будильника, во время выполнения задания, на экране успешного завершения и в важных шагах настройки разрешений.

Для показа рекламы используется Рекламная сеть Яндекса. Рекламный SDK может обрабатывать технические сведения об устройстве и приложении, сетевые сведения, рекламный идентификатор Android, данные о загрузке, показе и взаимодействии с рекламой, а также диагностические данные.

3. Техническая аналитика

При согласии пользователя приложение использует AppMetrica для технической аналитики, диагностики стабильности и улучшения качества.

Приложение не передает в AppMetrica точное время будильника, текст напоминания, QR-данные, выбранную мелодию, контакты, SMS, фотографии, микрофон и точное местоположение.

Пользователь может отключить отправку аналитики в настройках приложения.

4. Разрешения Android

Приложение запрашивает разрешения только для заявленных функций:

  • уведомления — для активного будильника;
  • точные будильники — для срабатывания в выбранное время;
  • экран поверх блокировки — для показа экрана звонка;
  • камера — только для сканирования QR-кода в QR-задании;
  • физическая активность — только для задания с шагами;
  • датчики движения — только для задания со встряхиванием;
  • интернет и состояние сети — для рекламы и технической аналитики.

5. Передача третьим лицам

Разработчик не продает пользовательские данные.

Данные могут обрабатываться третьими лицами только через используемые SDK:

  • AppMetrica — техническая аналитика при согласии пользователя;
  • Рекламная сеть Яндекса — показ рекламы и диагностика рекламных блоков.

6. Хранение и удаление данных

Локальные данные хранятся на устройстве до удаления приложения, очистки данных приложения в настройках Android или сброса статистики внутри приложения.

7. Права пользователя

Пользователь может:

  • отключить техническую аналитику;
  • отозвать разрешения Android в системных настройках;
  • сбросить статистику пробуждений;
  • удалить приложение и локальные данные;
  • обратиться по вопросам обработки данных на svetaet.support@gmail.com.

8. Изменения политики

Разработчик может обновлять настоящую Политику. Актуальная версия публикуется на этой странице.
